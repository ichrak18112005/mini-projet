\# Projet – Application Client/Serveur DevOps



\## Contenu du dépôt

\- client\_v1 : première version simple.

\- client\_v2 : version avancée.

\- serveur\_v1 : première version simple.

\- serveur\_v2 : version avancée.



\## Lancer sans Docker

python serveur\_v2/serveur2.py

python client\_v2/client2.py



\## Lancer avec Docker

docker-compose up



\## Fonctionnalités :

\- Ajouter une tâche

\- Supprimer une tâche

\- Modifier statut

\- Lister les tâches

\- Connexions multi-clients
